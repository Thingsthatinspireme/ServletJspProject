### 뜨게질 커뮤니티
1.기간 : 2021.01.26-2021.02.01
2.환경 : Apache Tomcat 8.0
3.주제 : 게시판기능이 있는 뜨게질 커뮤니티 사이트
4.개발 : STS, MySQL
5.사용언어 : JAVA, MySQL, JSP/Servlet
  (프론트는 무료 템플릿을 수정하고 추가해서 사용)

#### 구현 기능
1. 댓글 CRUD 기능
2. 게시글 조회 수
3. 게시물에 달린 댓글 수
4. 메인페이지의 인기글 모아보기
